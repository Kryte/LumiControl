Poor lighting on video calls? LumiControl turns your monitor into a soft, adjustable light source. Fine-tune the brightness and color temperature to get the perfect lighting for your webcam.

## Features

-   Toggle the light on/off.
-   Adjustable brightness slider.
-   Adjustable color temperature slider (from warm to cool).

## How to Use

1.  Download or clone the repository.
2.  Open the `lumi-control.html` file in your web browser.
3.  Position the browser window on the screen you wish to use as a light source to illuminate your face.

## Technologies Used

-   HTML5
-   CSS3
-   JavaScript (no frameworks)
